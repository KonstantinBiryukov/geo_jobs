Visualization of demand for software developer jobs regionally in the US.

About the project:

The demand for different programming languages has varied throughout the last few decades, and this variability is distinguished not only by skill set but by geographical fluctuations across the states. Inspired by a lack of tools to visualize demand for programming job skills by geographic region, this project seeks to create a web application that would pull job listings and add them to a map displaying demand for jobs regionally in the US, namely demand for specific skills. Besides, this project illustrates another use case, visualization of all US job locations from 5 large software companies: Facebook, Amazon, Apple, Netflix, Google.
“The power of maps comes from their inherent ability to leverage schemas that your users have been building up for many years”, Data visualization designer, Ryan Sleeper, Practical Tableau. The power of the map is that it can be quickly interpreted by the audience. Graphics and diagrams are not too illustrative when it deals with any geographical distribution. A map allows to distinguish any patterns just in several seconds, especially when we use clusters instead of individual points on the map. This way we can identify areas which has a high density of a chosen variable, in my case a variable is jobs of software development companies.

Techonology stack:

Python, HTML5, CSS3, JavaScript, Bootstrap 4, Mapbox, D3.js
